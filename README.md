# Closira AI Agent Assignment

## Features

- SOP-grounded FAQ answering
- Lead qualification workflow
- Escalation detection
- Conversation summarization

## Tech Stack

- Python
- Groq API
- Llama 3.1
- VS Code

## Setup Instructions

### Install Dependencies

pip install openai python-dotenv

### Add .env File

```env
GROQ_API_KEY=your_key_here
```

### Run Project

python main.py

## Escalation Triggers

- Angry sentiment
- Complaints
- Medical questions
- Human support requests
- Pricing negotiation

## Limitations

- Keyword-based escalation logic
- CLI-based interface
